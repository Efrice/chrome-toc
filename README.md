# TOC

Chrome extension to generate TOC for the current page.

## Features

- 🍭 TOC(expandable and draggable).
- 🧂 Easy to close.
- 🌝 Dark mode.

## Usage

1. Download the repo.
2. Open Manage Extends(chrome://extensions/) -> open Developer mode -> load unpacked -> select the directory of repo.

✨ Happy hacking!
